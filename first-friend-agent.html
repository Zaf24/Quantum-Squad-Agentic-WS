Welcome, Repair Specialist

Welcome, Repair Specialist.

Your mission—should you choose to accept it—is to master the art of building the First Friend Agent using Microsoft Copilot Studio.

This hands-on training is your entry point into the world of intelligent onboarding repair: from natural conversations to Adaptive Cards and agent flows, you'll learn how to build, deploy, and optimize an agent that can handle unexpected Day 1 scenarios like Alex's manager being on leave.

Repair Objectives

By completing the Engine Repair training, you'll be able to:

  • Understand how the First Friend Agent fits into the overall onboarding process repair
  • Explore how Large Language Models (LLMs), conversation flows, and Teams integration work together in an agent
  • Build a conversational agent that can handle Day 1 emergencies
  • Enhance the agent with Topics and Agent Flows
  • Deploy the agent to Microsoft Teams for immediate Alex support
  • Create autonomous decision-making capabilities for unexpected scenarios

Prerequisites

To complete the Engine Repair, you'll need:

  • A Microsoft 365 Developer tenant (with SharePoint enabled)
  • Access to Microsoft Copilot Studio (trial or licensed)
  • Microsoft Teams access for testing and deployment
  • Optional: Basic knowledge of SharePoint, Power Platform, or conversational AI concepts

Who This Is For

This repair training is ideal for:

  • HR professionals looking to modernize onboarding processes
  • IT pros building Microsoft 365 Copilot extensions for employee experience
  • Power Platform enthusiasts who want to level up with intelligent agents
  • Anyone who prefers to learn by solving real-world problems
  • Teams administrators wanting to improve new hire experience

🚨 Alex's Emergency Scenario

Remember: Alex arrives on his first day to discover his manager is unexpectedly on leave. The traditional onboarding process breaks down immediately. Your First Friend Agent must step in as the emergency repair system, ensuring Alex feels welcomed, connected, and supported despite the unexpected situation.

Repair Curriculum Overview

This repair training is broken into progressive lessons—each one designed as a repair step to get Alex's onboarding back to 100% operational efficiency.

Step Repair Task Repair Briefing
00 🔧 Workshop Setup Set up your repair environment, Copilot Studio trial, and Teams workspace
01 🛠️ Copilot Studio Fundamentals Learn the building blocks: knowledge, skills, and conversation flows
02 👋 Create the First Friend Agent Build the core agent that will welcome and support Alex on Day 1
03 🧩 Package Your Solution Create a reusable solution for deploying across different environments
04 🚀 Emergency Response Topics Add Topics to handle specific Day 1 scenarios and manager absence
05 ✍️ Natural Conversation Flows Create conversational paths that feel natural and supportive to Alex
06 🧠 Smart Triggers Use intelligent triggers to detect when Alex needs help
07 🪪 Team Connection Cards Build Adaptive Cards to help Alex connect with team members
08 🔁 Automated Team Setup Use Agent Flows to automatically create Teams groups and connections
09 🧭 Proactive Support Enable your agent to act autonomously and proactively support Alex
10 📢 Deploy to Teams Deploy your First Friend Agent to Microsoft Teams for Alex to use
11 🪪 Testing & Validation Test your agent with Alex's scenario and validate the repair effectiveness
