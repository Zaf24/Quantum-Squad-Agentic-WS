Workshop Overview

Welcome to the Quantum Squad Workshop Setup! Before we can start repairing Alex's onboarding process, we need to set up our repair tools and environment. This guide will walk you through the process of setting up your Microsoft Copilot Studio trial and creating a development environment where you'll build your repair agents.

Just like a mechanic needs their tools and workspace ready before fixing a car, you'll need to set up your digital repair shop before building your onboarding repair agents. This one-time setup will give you everything you need to build powerful, intelligent agents using Microsoft Copilot Studio.

Prerequisites

Before you begin, make sure you have:

  • A work or school Microsoft account (personal accounts won't work for this workshop)
  • Administrator permissions on your Microsoft 365 tenant (to create the developer environment)
  • A modern web browser (Microsoft Edge, Google Chrome, or Safari)
  • Approximately 20 minutes to complete the setup process
Note: If you don't have a work/school Microsoft account with admin permissions, you can create a Microsoft 365 Developer tenant for free at Microsoft 365 Developer Program.

Step 1: Sign in to your work/school Microsoft account

First, we'll set up your Microsoft Copilot Studio trial:

  1. 1.1 Go to https://www.microsoft.com/en-us/microsoft-365-copilot/microsoft-copilot-studio
  2. 1.2 Select "Try for free"
  3. 1.3 Enter your work email account
  4. 1.4 It should recognize your account. Select "Sign In"
  5. 1.5 Select "Start free trial"

Step 2: Create a developer environment

Using the same Microsoft 365 tenant from Step 1, we'll now set up a Power Apps Developer Plan to create a free development environment for building and testing with Copilot Studio:

  1. 2.1 Using the same Microsoft 365 tenant in Step 1, sign up for a Power Apps Developer Plan to create a free development environment to build and test with Copilot Studio.
  2. 2.2 Go to https://www.microsoft.com/en-sg/power-platform/products/power-apps/free
  3. 2.3 Click "Start Free"
  4. 2.4 Enter your email address, tick the checkbox and click "Start free"
  5. 2.5 After signing up for the Developer Plan, you'll be redirected to Power Apps. The environment uses your name, for example Adele Vance's environment. If there's already an environment with that name, the developer new environment is named Adele Vance's (1) environment. Use this developer environment in Copilot Studio when completing the labs.
